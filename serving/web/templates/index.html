Sign in with Angel One

You'll be taken to Angel One's own login page. Enter your client code, PIN and TOTP there — then you'll come straight back here.

This app doesn't run its own registered SmartAPI application by default — you bring your own. It's free and takes a minute: register an app at smartapi.angelone.in, copy the API key it gives you, and paste it above. Nothing else from that page (no secret, no credentials) is needed here. This deployment also has a shared default key configured, so you can leave this field blank if you'd rather use that instead.

🔒 What we never see

  • ✓ Your PIN — typed on angelone.in, not here
  • ✓ Your TOTP secret — never requested, never stored
  • ✓ Your password — it never reaches this server

What this app can do

  • ✓ Read your holdings, to analyse them
  • ✗ Place, modify or cancel orders — no such code exists in this app
  • ✗ Move money or withdraw funds

Your session lives on the server only and ends at midnight IST, or when you sign out. Your browser holds nothing but an opaque session id. The API key you enter above travels once, with this login attempt only — it isn't stored anywhere after your session ends.

No Angel One account?

You can still analyse your mutual funds — enter schemes manually or upload a CAS.
